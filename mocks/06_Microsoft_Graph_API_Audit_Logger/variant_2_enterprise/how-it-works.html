Log Collection & Processing

GraphAudit Enterprise uses Microsoft Graph API and Azure Management API to collect audit logs across all M365 services. Multi-threaded collection ensures no event loss even during traffic spikes.

  • Collection Rate: 100M+ events/day with auto-scaling
  • API Integration: Graph API, Exchange Admin API, SharePoint API, Azure AD Graph
  • Event Types: 240+ event types across Exchange, SharePoint, Teams, OneDrive, Azure AD, Intune, Defender
  • Processing Latency: Sub-5-second from event occurrence to indexed log
  • Reliability: Automatic retry with exponential backoff, zero data loss guarantee

Storage & Retention

Logs stored in geo-redundant, immutable storage with configurable retention policies. Option to use your own Azure Storage or AWS S3 buckets (BYOS - Bring Your Own Storage).

  • Storage Options: GraphAudit-managed (Azure Blob) or customer-managed (S3, Azure, GCS)
  • Retention: Unlimited retention with tiered storage (hot/cool/archive) for cost optimization
  • Encryption: AES-256 at rest, TLS 1.3 in transit. Customer-managed keys (BYOK) supported
  • Immutability: WORM (Write Once, Read Many) compliance for regulatory requirements
  • Replication: Multi-region replication for disaster recovery (RPO: <1hr, RTO: <4hr)

Search & Analytics

Real-time search with sub-second latency powered by Elasticsearch cluster. Advanced query language for complex investigations.

  • Search Performance: Query 5+ years of logs in <1 second
  • Query Language: Lucene-based syntax with regex, wildcards, boolean operators
  • Analytics: Pre-built dashboards (Grafana), custom visualizations, trend analysis
  • Reporting: Scheduled reports (daily/weekly/monthly), export to CSV/JSON/PDF
  • API Access: REST API for programmatic access, GraphQL support

Security & Compliance

Zero-trust architecture with defense-in-depth strategy. Automated compliance controls for SOC 2, HIPAA, FedRAMP.

  • Read-Only API Access: No write permissions to your M365 tenant. Metadata-only logging (no email bodies, file contents)
  • Data Residency: Choose storage region (US, EU, APAC, Middle East). GDPR-compliant data processing
  • Access Control: SSO (SAML 2.0), RBAC with custom roles, MFA enforcement, IP whitelisting
  • Audit Trail: We log who accessed audit logs (meta-logging) for forensic accountability
  • Certifications: SOC 2 Type II, HIPAA, FedRAMP Moderate, ISO 27001, ISO 27017, ISO 27018
