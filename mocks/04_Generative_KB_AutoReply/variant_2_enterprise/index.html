Layered Microservices Architecture:

1. Ingestion Layer: Real-time connectors to knowledge sources (REST APIs, webhooks, SFTP). Document parsing, OCR, metadata extraction.

2. Processing Layer: Chunking, vectorization (OpenAI embeddings, sentence-transformers), semantic indexing (Pinecone, Weaviate, Qdrant).

3. Generation Layer: LLM orchestration (LangChain, LlamaIndex), prompt engineering, response synthesis with citations.

4. Delivery Layer: ITSM webhook handlers, email SMTP, Slack/Teams bots, REST API for custom integrations.

5. Feedback Loop: Agent thumbs-up/down, A/B testing, reinforcement learning from human feedback (RLHF).

Deployed on Kubernetes (AWS EKS, Azure AKS, GCP GKE) or on-prem air-gapped environments.
